Repeat()

Returns a IndexedSeq of value repeated times times. When times is not defined, returns an infinite Seq of value.

Repeat<T>(value: T, times?: number): IndexedSeq<T>

Example

Repeat('foo') // ['foo','foo','foo',...] Repeat('bar',4) // ['bar','bar','bar','bar']
